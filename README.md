# climarch
Data, code and analysis of the Climate Risk Assessment for the Brazilian Cultural Heritage project  

O objetivo do projeto é investigar como museus e outros patrimônios culturais brasileiros estão expostos a riscos climáticos, ajudando a priorizar recursos de preservação e desenvolver estratégias de salvaguarda que protejam não apenas o patrimônio material, mas as histórias e identidades que eles as sustentam.

- **Que porcentagem dos nossos principais museus está em áreas de alto risco climático**
- **Que memórias e identidades regionais podem ser perdidas se essas instituições forem afetadas?**
- **Os acervos de maior risco são os mesmos em todo o Brasil, ou cada região enfrenta ameaças específicas ao seu patrimônio único?**
